NOIRE

NOIRE is a premium menswear e-commerce platform designed to deliver a seamless shopping experience through modern design, intuitive navigation and secure purchasing workflows. The project combines UX design, visual design, front-end development and product thinking into a complete end-to-end digital product.

Project Overview

Role

UI/UX Designer, Product Designer & Front-End Developer

Duration

2026

Tools

Figma, Photoshop, HTML, CSS, JavaScript

Stack

Node.js, Express, MongoDB, Razorpay

Project Highlights

  • Authentication System
  • Wishlist Functionality
  • Shopping Cart & Checkout
  • Razorpay Payment Integration
  • Order Management
  • Admin Dashboard
  • Stock Management System
  • Responsive Design

The Problem

Many online fashion stores overwhelm users with cluttered interfaces, inconsistent product browsing and complicated checkout experiences. The challenge was to design a platform that feels premium while keeping shopping simple, intuitive and frictionless.

Goals

  • Create a premium fashion shopping experience
  • Build a complete e-commerce workflow
  • Implement secure authentication
  • Integrate online payments
  • Create an admin management system
  • Maintain a modern and minimal design language

Project Outcomes

  • Designed and developed a complete e-commerce platform from scratch
  • Integrated secure authentication and payment workflows
  • Built product, wishlist, cart and order management systems
  • Created a responsive experience across desktop and mobile devices

Design Process

The project began with researching modern fashion brands and analysing common e-commerce user flows. Wireframes were created to define navigation, product discovery and checkout experiences before moving into high-fidelity UI design and development.

Key Features

  • User Authentication & Secure Login
  • Wishlist Management
  • Shopping Cart System
  • Dynamic Product Pages
  • Order Tracking
  • Razorpay Payment Gateway
  • Admin Product Management
  • Inventory & Stock Control
  • Responsive Mobile Experience

Challenges

One of the biggest challenges was connecting the frontend and backend into a seamless user experience. Special attention was required for authentication, cart management, order processing, stock updates and payment verification while maintaining a clean and premium user interface.

Final Product

NOIRE evolved into a complete fashion commerce ecosystem, allowing users to browse products, manage wishlists, complete purchases securely and track orders while administrators manage inventory and products through a dedicated dashboard.

Key Learnings

Through NOIRE, I strengthened my understanding of product design, user flows, frontend development, backend architecture and payment integrations. The project helped me understand how design and development work together to create complete digital products from concept to deployment.
